How to build and maintain this blog

June 24, 2023

1. Workflow#

  1. Add posts by hugo new posts/post_name.md
  2. git add . git commit -m ""
  3. Generate the site by hugo
  4. cd public git add . git commit -m "" git push

Run hugo server and view website locally at localhost:1313
